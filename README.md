A Map::Tube implementation for the Toulouse tube network.

This module allows to find the shortest route between any two given tube, tram,
and funicular stations in Toulouse.

All interesting methods are provided by the role L<Map::Tube>.

To build this module, use the classical steps:

* perl Makefile.PL
* make
* make test
* make install

(If you are using Srawberry Perl under Windows, you may want to replace "make"
with "gmake".)
